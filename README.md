## ssh博客系统（从前端到后端实现）
### 开发环境
 * window7
 * eclipse： Mars.2 Release (4.5.2）
 * Mysql 5.5.1
 * jetty9.2.2
 * maven4.0
 * Sublime Text3

### 框架版本
* Spring4.3.4
* SpringMVC4.3.4
* Hibernate4.3.11

### 实现功能
1. 前端
* 前端页面设计，排版
* logo设计
* psd切图，转化成标准HTML5,CSS3页面
* 参考Bootstrap的栅格系统，实现响应式布局
* ajax/json实现分页和请求数据

2. 后端
* 引入showdown.js,实现markdown写作支持
* 用户登录、注册，信息修改
* 博文的增删改查、分类、分页
* 评论的删除、分页、管理。
* 作品的增删改查、上传图片、删除图片
